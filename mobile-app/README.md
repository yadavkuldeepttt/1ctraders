# 1C Traders Mobile App

React Native mobile application for 1C Traders investment platform.

## Features

- Investment management
- Referral system (12 levels)
- Task system for earning rewards
- Crypto wallet integration
- Real-time balance updates
- Transaction history

## Setup

1. Install dependencies:
```bash
npm install
```

2. Start the development server:
```bash
npm start
```

3. Run on Android:
```bash
npm run android
```

4. Run on iOS:
```bash
npm run ios
```

## Requirements

- Node.js 18+
- Expo CLI
- Android Studio (for Android development)
- Xcode (for iOS development, macOS only)
